# Calculadora de Propinas con React y Tailwind CSS

Este proyecto es una aplicación web que permite calcular propinas de una manera fácil y rápida. Está construido con React y utiliza Tailwind CSS para el diseño y estilos.

## Características

- **Calculadora de Propinas**: Ingresa el monto de la factura, selecciona el porcentaje de propina y la cantidad de personas para obtener el monto total a pagar por cada persona.

- **Diseño Responsivo**: La aplicación está diseñada para funcionar en dispositivos de diferentes tamaños, desde teléfonos móviles hasta escritorios.

## Tecnologías Utilizadas

- **React**: Biblioteca de JavaScript para construir interfaces de usuario interactivas.

- **Tailwind CSS**: Framework de utilidad CSS de baja abstracción para diseñar rápidamente interfaces de usuario.

## Instalación

1. Clona este repositorio: `git clone https://github.com/roddcode/tip-calculator-app-main.git`

2. Navega al directorio del proyecto: `cd ./tip-calculator-app-main`

3. Instala las dependencias: `npm install`

4. Inicia la aplicación: `npm start`

5. Abre tu navegador y visita `http://localhost:3000` para ver la aplicación en acción.
